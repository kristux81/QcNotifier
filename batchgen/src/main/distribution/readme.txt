Batch Generator for TDNotifier Batch Client
--------------------------------------------


Requirements
-------------
-Java 32 or 64 Bit JRE 1.6 or later


Supported Platforms
---------------------
- Linux / UNIX ( those supporting SUN Java )
- Windows ( those supporting JRE 6 )


Command
-------

java -jar lib/qc-notifier-batchgen-*.jar <input_file> <map_file>
